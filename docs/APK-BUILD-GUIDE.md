# IDEAGO Android APK Build Guide

## Prerequisites

| Tool | Version | Download |
|------|---------|----------|
| Node.js | 18+ | https://nodejs.org |
| Android Studio | Latest | https://developer.android.com/studio |
| Java JDK | 17 | Bundled with Android Studio |

## Android Studio Setup (One-Time)

1. **Install Android Studio** from https://developer.android.com/studio

2. **Install SDK components** (Android Studio > Settings > SDK Manager):
   - Android SDK Platform 34 (Android 14)
   - Android SDK Build-Tools 34
   - Android SDK Command-line Tools
   - Android Emulator (optional, for testing)

3. **Set environment variables** (add to `~/.bashrc` or `~/.zshrc`):
   ```bash
   export ANDROID_HOME=$HOME/Android/Sdk
   export PATH=$PATH:$ANDROID_HOME/platform-tools
   export PATH=$PATH:$ANDROID_HOME/cmdline-tools/latest/bin
   ```
   On macOS: `$HOME/Library/Android/sdk`
   On Windows: `%LOCALAPPDATA%\Android\Sdk`

4. **Accept licenses**:
   ```bash
   sdkmanager --licenses
   ```

## Quick Build (Automated Script)

```bash
./scripts/build-apk.sh
```

The script handles: dependency check → Next.js build → Capacitor sync → Gradle build.

## Manual Build Steps

### Step 1: Install Dependencies

```bash
cd frontend
npm install
```

### Step 2: Add Android Platform (First Time Only)

```bash
npx cap add android
```

This generates `frontend/android/` — a full Android Studio project.

### Step 3: Build & Sync

```bash
npm run cap:sync
```

This runs `next build` (static export to `out/`) then copies web assets to the Android project.

### Step 4: Build APK

**Option A — Android Studio (Recommended for first build):**
```bash
npm run cap:open
```
Then in Android Studio: **Build > Build Bundle(s) / APK(s) > Build APK(s)**

**Option B — Command Line:**
```bash
cd android
./gradlew assembleDebug
```

Debug APK location: `android/app/build/outputs/apk/debug/app-debug.apk`

### Step 5: Install on Device

```bash
adb install android/app/build/outputs/apk/debug/app-debug.apk
```

Or transfer the APK file to your Android device and install manually.

## Build Types

| Type | Command | Use Case |
|------|---------|----------|
| Debug | `./gradlew assembleDebug` | Development & testing |
| Release | `./gradlew assembleRelease` | Production (requires signing) |
| Bundle | `./gradlew bundleRelease` | Google Play Store upload |

## Release Signing (Production)

1. **Generate keystore**:
   ```bash
   keytool -genkey -v -keystore ideago-release.keystore \
     -alias ideago -keyalg RSA -keysize 2048 -validity 10000
   ```

2. **Configure signing** in `android/app/build.gradle`:
   ```gradle
   android {
       signingConfigs {
           release {
               storeFile file('ideago-release.keystore')
               storePassword System.getenv('KEYSTORE_PASSWORD')
               keyAlias 'ideago'
               keyPassword System.getenv('KEY_PASSWORD')
           }
       }
       buildTypes {
           release {
               signingConfig signingConfigs.release
           }
       }
   }
   ```

3. **Build signed APK**:
   ```bash
   KEYSTORE_PASSWORD=xxx KEY_PASSWORD=xxx ./gradlew assembleRelease
   ```

## Troubleshooting

### `out/` directory not created
Ensure `next.config.js` has `output: "export"`. Server-side features (`getServerSideProps`, API routes, middleware) are incompatible with static export.

### White screen on app launch
Check that `webDir` in `capacitor.config.ts` is `"out"` and the build completed successfully.

### API calls failing
The app runs locally on the device. Ensure API calls point to an accessible server URL, not `localhost`. Update `NEXT_PUBLIC_API_URL` to the production API endpoint before building.

### Capacitor plugin not found
Run `npx cap sync android` after installing new Capacitor plugins.

### Gradle build fails with SDK error
Verify `ANDROID_HOME` is set and the required SDK platform is installed via Android Studio SDK Manager.
