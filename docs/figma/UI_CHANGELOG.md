# IDEAGO KEY FEATURE SUMMARY

## 문서 목적
이 문서는 피그마 작업 이후 현재 기준으로 실제 구현에 반영할 핵심 기능만 정리한 문서다.

## 핵심 기능

1. 새 프로젝트 생성
- 프로젝트명 수동 입력 가능
- 자동 제목 생성 가능

2. 에이전트 구성
- 에이전트 수 선택 가능
- 에이전트 역할 커스터마이징 가능

3. 메인 캔버스
- 자유 드로잉
- 텍스트 입력
- 도형 입력
- 색상 변경
- 선택/이동/복사/자르기

4. 이미지 업로드 + 마킹
- 사진/이미지 업로드
- 이미지 위에 직접 스케치 및 메모 가능

5. 에이전트 결과창
- 사용자 질문 입력
- 에이전트 요약형 응답 출력
- 상세보기 선택 가능

6. Detail View
- 선택적으로 열 수 있는 결과 시각화 기능
- 항상 열려 있는 구조는 아님

7. 프로젝트 요약/히스토리
- 프로젝트 정보
- 작업 요약
- 에이전트 구성 요약

8. Export
- 구조화된 폴더/ZIP 패키지 내보내기
- 요약문서, 에이전트 정보, 캔버스 데이터, 업로드 자료 포함

9. 반응형 대응
- 태블릿 가로
- 태블릿 세로
- 모바일 세로 9:16 대응

## 구현 원칙
- UI는 바뀔 수 있지만 기능 책임은 유지한다.
- 시각화는 플랫폼 코어 기능이다.
- 에이전트는 보조 역할이며 기본은 요약형 응답이다.
- 사용자 행위는 구조화 데이터로 저장되어야 한다.

# UI Changelog
arkistore.com 스타일로 전체 디자인을 리뉴얼. 깔끔한 미니멀리즘, 넓은 여백, 세련된 타이포그래피, 고급스러운 컬러 팔레트로 변경.

모든 화면의 디자인을 arkistore.com 스타일로 리뉴얼 완료. 주요 변경사항은 다음과 같습니다:

전체 디자인 시스템을 미니멀하고 세련된 방향으로 전면 교체했습니다. 다크 톤(#1D1D1F) 기반의 프라이머리 컬러, 넓은 여백과 라운드 2xl 카드, Apple HIG 스타일의 uppercase 영문 섹션 라벨, 얇은 아이콘(strokeWidth 1.5), 호버/액티브 마이크로인터랙션, 고급스러운 그라데이션 히어로 배너, backdrop-blur 네비게이션 등 arkistore.com의 에디토리얼 감성을 반영했습니다. 기존 블루 컬러(#2563EB)는 예산/금액 등 강조 포인트에 액센트로 활용됩니다.
